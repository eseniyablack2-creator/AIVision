# Шаг 1

## Что делаем прямо сейчас

Мы создаем каркас приложения.

Так как в корне проекта уже лежат документы, клиентская часть создается в подпапке `frontend`.

## Что должно получиться

После первого шага у нас будет:
- локальный проект;
- команда запуска;
- пустой интерфейс с заделом под DICOM viewer.

## Что я рекомендую как первый технический шаг

Создать приложение на:
- React;
- TypeScript;
- Vite.

## Почему именно так

- это быстрее и стабильнее, чем старый Create React App;
- TypeScript поможет раньше замечать ошибки;
- Vite проще для локальной разработки и быстрее стартует.

## Команды первого шага

Открыть терминал в папке проекта и выполнить:

```powershell
npm create vite@latest frontend -- --template react-ts
cd frontend
npm install
npm run dev
```

## Ожидаемый результат

- в терминале появится локальный адрес приложения;
- в браузере откроется стартовая страница Vite;
- после этого можно переходить к подключению DICOM-движка.
